Overview of the app

This app was built based on mtcars dataset. The data was extracted from the 1974 Motor Trend US magazine, and comprises fuel consumption and 10 aspects of automobile design and performance for 32 automobiles. This app allows the regression analysis of the dataset using the following tools:

  • Box plot of the outcome vs one predictor
  • Regression model of outcome vs a number of predictors chosen by the user
  • Diagnostics plots (Residuals vs Fitted, Normal Q-Q, Scale-Location, Residuals vs leverage)

Box plot tab

library(ggplot2)
chkgrp<- "am" #This variable stores the checkbox ("am" is checked by default)
outcome<- "mpg" #This variable stores the outcome selected in the textbox
                      # ("mpg" by default)
ggplot(data = mtcars, aes(x = cut(mtcars[,chkgrp[1]],5), y = mtcars[,outcome], 
                          .groups = 'keep')) + geom_boxplot()+
    xlab(chkgrp[1]) + ylab(outcome)+ theme(axis.text=element_text(size=12),
          axis.title=element_text(size=14,face="bold"))
